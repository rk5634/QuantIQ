# AI-Driven Stock Market Analysis and Prediction

This project implements a machine learning pipeline for stock market analysis and prediction. It predicts Buy/Sell/Hold signals based on technical indicators.

## Structure

- `config/`: Configuration settings.
- `data/`: Data storage (raw and processed).
- `logs/`: Application logs.
- `src/`: Source code modules.
- `tests/`: Unit tests.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the pipeline:
   ```
   python main.py --data_path data/raw/your_data.csv
   ```
